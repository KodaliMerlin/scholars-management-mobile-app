# Scholar_management_mobile_application

A new Flutter project. Designed for the Foundation Empower Ananya, which is a Girl Scholars Foundation that supports Tech Girls by providing various facilities like Study, Scholarships, Internships, Mentorships and What not. In order to maintain the Scholars track and observe their performance in assigned courses or internship,etc., this app was specifically designed for EmpowerAnanya Foundation Manager.

## Getting Started

This project is a starting point for a Flutter/Firebase application.
Built on various means like
- Scholar Performance Metrics tracking
- Adding new programs and Scheduling
- Sending reminders through mails and whatsapps
- Storing new events and internships directly on firestore
- An automated app scripts from a Google Form when new shcolar fills it to add onto the Foundation, etc.
